# Raycast 启动 Claude Code

在 Finder 当前目录下快速启动 Claude Code。

## 前提

- [Raycast](https://raycast.com/) 已安装
- 任意 AI 编程工具已安装（如 Claude Code、Cursor、Windsurf 等）

## 使用

1. 把脚本放到 Raycast Scripts 目录
2. 在 Raycast 中搜索脚本（输入 "Launch Claude Code"）
3. 按回车运行

## 配置快捷键

1. 打开 Raycast，按 `Cmd + ,` 进入 Settings
2. 点击左侧 **Extensions**
3. 搜索 "Launch Claude Code"
4. 点击 **Add Shortcut**，设置你喜欢的快捷键

## 用 AI 修改

把下面的 prompt 复制到 Claude Code，让它帮你改成你用的终端：

```
我想要修改 Raycast 启动 Claude Code 的脚本，请帮我把终端改成 iTerm2。
脚本在 [path/to/]launch-claude-code.sh
```

把 `iTerm2` 换成你用的终端即可。
